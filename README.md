# Valheim AI Bot RL Trainer

Training AI model to play Valheim with reinforcement learning.

## Info

- valheim_ai_v1.py
  - Basic training with limited, random actions; move forward, backward, left, right, attack, and jump.
- valheim_ai_v2.py
  - Improved training with additional actions and rewards/penalties.
